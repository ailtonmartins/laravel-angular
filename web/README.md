WEB - ANGULA 6 + ANGULAR MATERIAL
=================

Requisito
---------------
Ter instalado angular/cli = 6

Instalação
---------------
Após clonar o projeto, voce vai entrar no repositório WEB e rodar o comando:
```
npm install
``` 

Rodar o servidor com o comando 
``` 
ng serve
```
Vamos testar 
  
 [http://localhost:4200/](http://localhost:4200/)
